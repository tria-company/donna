## How to Use This KB

1. Read the relevant file(s) based on the user's question
2. If unsure which file, use `Grep` to search across all files: `~/.claude/hormozi-kb/*.md`
3. Multiple files often apply — cross-reference for complete answers
4. The Value Equation from `100m-offers.md` underpins almost everything
5. Playbooks are the most actionable — use them for specific tactical advice
6. Blackbooks show how Hormozi actually executes (not theory, real operations)
